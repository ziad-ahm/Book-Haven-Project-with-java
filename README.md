# 📚 Book Haven Project

A full-stack web application built using **Spring Boot** for the backend and **HTML/CSS** for the frontend.

The project started as a static frontend website and was later integrated with a Java Spring Boot backend to make it dynamic and scalable.

---

## 🚀 Features

- 🏠 Home page
- 📖 Books page
- 📘 Book details pages
- 🔐 Login page UI
- 📦 Track order page
- 🎨 Responsive design using HTML & CSS
- ⚙️ Backend powered by Spring Boot

---

## 🛠️ Tech Stack

### 🔹 Backend
- Java
- Spring Boot
- Maven

### 🔹 Frontend
- HTML5
- CSS3
